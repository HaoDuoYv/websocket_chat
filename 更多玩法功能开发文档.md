# 更多玩法功能开发文档

## 1. 功能概述

本功能旨在为聊天应用添加一个"更多玩法"模块，用户可以通过点击导航栏中的玩法图标进入玩法列表页面，查看和参与各种互动玩法。

## 2. 技术架构

### 2.1 前端架构
- **框架**：Vue 3 + TypeScript
- **路由**：Vue Router
- **样式**：Tailwind CSS
- **状态管理**：组合式 API

### 2.2 后端架构
- **框架**：Spring Boot
- **API**：RESTful API
- **WebSocket**：实时通信

## 3. 功能设计

### 3.1 导航栏集成
- 在左侧导航栏添加"玩法"图标
- 点击图标切换到玩法列表页面
- 保持与现有导航风格一致

### 3.2 玩法列表页面
- 显示玩法分类
- 展示玩法卡片
- 支持玩法搜索
- 保持与现有UI风格一致

### 3.3 玩法详情页面
- 展示玩法规则
- 提供参与入口
- 显示玩法状态

### 4.2 后端实现

#### 4.2.1 玩法 API 接口

**文件**：`/workspace/websocket_chat/backend/src/main/java/com/chat/controller/GameController.java`

```java
package com.chat.controller;

import org.springframework.web.bind.annotation.*;

import java.util.Arrays;
import java.util.List;

@RestController
@RequestMapping("/api/games")
@CrossOrigin(origins = "*")
public class GameController {

    @GetMapping("/list")
    public List<GameInfo> getGames() {
        return Arrays.asList(
            new GameInfo("1", "猜数字", "casual", "猜数字游戏，猜对即可获得奖励", "🎯", "available"),
            new GameInfo("2", "成语接龙", "social", "和好友一起玩成语接龙", "📚", "available"),
            new GameInfo("3", "你画我猜", "social", "画图猜词游戏", "🎨", "coming_soon")
        );
    }

    @GetMapping("/info/{gameId}")
    public GameInfo getGameInfo(@PathVariable String gameId) {
        // 根据游戏ID返回游戏详情
        return new GameInfo(gameId, "游戏详情", "casual", "游戏详细描述", "🎯", "available");
    }

    public static class GameInfo {
        private String id;
        private String name;
        private String category;
        private String description;
        private String icon;
        private String status;

        public GameInfo(String id, String name, String category, String description, String icon, String status) {
            this.id = id;
            this.name = name;
            this.category = category;
            this.description = description;
            this.icon = icon;
            this.status = status;
        }

        // getters and setters
        public String getId() { return id; }
        public void setId(String id) { this.id = id; }
        public String getName() { return name; }
        public void setName(String name) { this.name = name; }
        public String getCategory() { return category; }
        public void setCategory(String category) { this.category = category; }
        public String getDescription() { return description; }
        public void setDescription(String description) { this.description = description; }
        public String getIcon() { return icon; }
        public void setIcon(String icon) { this.icon = icon; }
        public String getStatus() { return status; }
        public void setStatus(String status) { this.status = status; }
    }
}
```

#### 4.2.2 前端 API 调用

**文件**：`/workspace/websocket_chat/frontend/src/api/games.ts`

```typescript
import axios from 'axios'

const API_BASE_URL = '/api'

export interface GameInfo {
  id: string
  name: string
  category: string
  description: string
  icon: string
  status: string
}

export const getGames = async (): Promise<GameInfo[]> => {
  const response = await axios.get<GameInfo[]>(`${API_BASE_URL}/games/list`)
  return response.data
}

export const getGameInfo = async (gameId: string): Promise<GameInfo> => {
  const response = await axios.get<GameInfo>(`${API_BASE_URL}/games/info/${gameId}`)
  return response.data
}
```
