# 协作编辑器

基于 Yjs CRDT 的实时协作代码编辑器，支持多人同时编辑 HTML/CSS/JS 代码，并实时预览渲染结果。

## 功能特性

- **实时协同**：基于 Yjs CRDT 算法，多人同时编辑无冲突
- **语法高亮**：CodeMirror 6 编辑器，支持 HTML/CSS/JS 语法高亮和自动补全
- **实时预览**：右侧 iframe 实时渲染 HTML/CSS/JS 代码结果
- **光标同步**：实时显示其他协作者的光标位置
- **多标签页**：HTML、CSS、JS 三个独立编辑标签页
- **主题切换**：支持亮色/暗色主题
- **撤销协作**：基于 Yjs 的协作感知撤销/重做

## 使用方式

1. 点击"前往"创建新文档
2. 分享页面 URL 给其他人
3. 多人同时编辑，变更实时同步
4. 右侧预览区查看渲染效果

## 技术栈

- 前端：Vue 3 + CodeMirror 6 + Yjs + TypeScript
- 后端：Spring Boot + WebSocket
- 协同算法：Yjs (CRDT)
