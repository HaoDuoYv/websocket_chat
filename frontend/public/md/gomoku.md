# 五子棋对战

基于 WebSocket 的实时五子棋对战游戏，支持创建房间、密码保护、观战、房间聊天。

## 功能特性

- **实时对战**：通过 WebSocket 实现低延迟的落子同步
- **房间系统**：创建/加入房间，支持密码保护
- **观战模式**：其他用户可进入房间观战
- **房间聊天**：对局中实时文字交流
- **断线重连**：意外断线后可重新加入对局
- **认输/再来一局**：完整的对局流程控制

## 使用方式

1. 进入应用后创建或加入房间
2. 等待两名玩家就位后点击"开始游戏"
3. 黑方先行，轮流落子，先连成五子者获胜

## 技术栈

- 前端：Vue 3 + TypeScript + Tailwind CSS + Canvas
- 后端：Spring Boot + WebSocket
- 通信：自定义 WebSocket 事件协议
